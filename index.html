Open-Ended 3D Metric-Semantic Representation Learning via Semantic-Embedded Gaussian Splatting

CVPR 2025 Submission

Anonymous Authors
Paper

Our framework turns an RGB-D stream into an open-ended metric-semantic representation.

Overview

TL;DR: We propose an open-ended metric-semantic representation learning framework based on 3D Gaussians, which distills open-set semantics from 2D foundation models into a scalable 3D Gaussian representation, optimized within a SLAM framework.

Abstract: This work represents one step towards utilizing everyday devices equipped with multi-view stereo to create a 3D virtual world. We propose an open-ended metric-semantic representation learning approach based on 3D Gaussians, which distills open-set semantics from 2D foundation models into a scalable and continuously evolving 3D Gaussian representation, optimized within a SLAM framework.Given the ever-expanding nature of 3D scenes, embedding semantic information directly into Gaussians becomes impractical, leading to excessive memory usage and semantic inconsistencies. In response, we propose to learn semantics by aggregating from a condensed, fixed-sized semantic pool rather than directly embedding high-dimensional raw features, reducing memory requirements compared to point-wise representation. Additionally, by enforcing pixel-to-pixel and pixel-to-object semantic consistency through contrastive and stability-guided optimization, our framework enhances coherence and stability in semantic representations. Experiments show that our framework yields a precise open-ended metric-semantic field with superior rendering quality and tracking accuracy, even surpassing SOTAs tailored for high-fidelity rendering. Besides, it enables the capture of both closed-set object categories and open-set semantics, facilitating various applications, notably fine-grained, unrestricted 3D scene editing. These results mark an initial yet solid step towards efficient and expressive 3D virtual world modeling.

Methodology

Framework Overview. (a) Each Gaussian is linked to a low-dimensional key to softly aggregate semantics from a fixed-size learnable feature pool. (b) The SLAM framework processes RGB-D frames with a semantic feature map generated by a pre-trained model. (c) To enhance semantic consistency and address ambiguity during optimization, we introduce the Intra-Inter Semantic Consistency Objective and Semantic Stability Guidance.
